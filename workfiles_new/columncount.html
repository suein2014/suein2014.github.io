Dividing text into columns

This div will have the text divided into 5 columns because of the journal class that is being used. It is interesting to notice the use of the column-count CSS property and the use of the vendor prefixes to make sure it will work in all browsers. The column-count still requires those vendor prefixes according to Caniuse.com website. In the future, you might be able to use this CSS property without prefix. If we do not have a big text, in the desktop/laptop, this text will be presenting only 4 columns but in the mobile devices, this text will be shown divided into 5 columns.
