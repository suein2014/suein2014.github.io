HTML5 Rocks!!!

This is a great example on how to make a background color gradient without the need to any graphical software. Simply using the gradient property of CSS3. You define the area - it could be the whole body of the HTML document or just a small space (a div like this one) - that you apply the gradient property and the browser will gradually change from one color to another color that you have determined in the CSS.

Here the gradient is being applied to work in all different browsers. Using the prefixes that are related to the browsers: -o for Opera; -ms for IE; -moz for Mozilla; and -webkit for Safari and Chrome. So, this gradient here should work in all types of browsers!!!
